# Rivet & Rail – Boutique Hotel Website

Rivet & Rail is a modern, luxury boutique hotel website designed to present the brand as an **urban sanctuary** in the heart of the city. The project focuses on delivering a clean, elegant user experience with a refined layout, contemporary typography, and sleek UI components.

This website highlights the hotel’s services, aesthetic identity, mission, and values while offering smooth navigation and a visually cohesive structure.

## Features

### Hero Section
- Large welcoming banner with tagline
- Call-to-action button: *Reserve Your Suite*

### Services Overview
- Icon-based grid showcasing:
  - Dining & Beverage Options
  - Fitness
  - Transportation & Parking
  - Laundry & Dry Cleaning
  - Movie Nights
  - Spa & Therapeutic Massages

### Aesthetic Showcase – “Contemporary Calm”
- Section highlighting the hotel’s modern, neutral interior design
- Photo + text layout for visual storytelling

### Client Testimonials
- Three review cards
- Highlights real guest experiences: comfort, service, and atmosphere

### About Section
- Brand story and philosophy
- “Our Mission” + “Bespoke Service” split text layout
- High-quality imagery

### Footer
- Logo and tagline
- Quick navigation links
- Address, phone, and email

## Tech Stack
- **HTML5**
- **CSS3**
- **Google Fonts**
- **Responsive Design**
- **SEO-ready metadata**
